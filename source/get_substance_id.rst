Function to retrieve the ID of a substance
==========================================

This function is used to retrieve the ID of a substance from the database based on its name.

Parameters:
-----------

- **substanceName**: The name of the substance.

Return Value:
-------------

- Returns the ID of the substance.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
