Function to retrieve the IDs of medication classes from a substance
==================================================================

This function is used to retrieve the IDs of medication classes associated with a given substance from the database.

Parameters:
-----------

- **substance**: The name of the substance.

Return Value:
-------------

- Returns a list of IDs of medication classes associated with the substance.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
