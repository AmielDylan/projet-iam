Importing mysql.connector
=========================

The ``mysql.connector`` module is a Python library that enables connection and interaction with a MySQL database from Python programs.

Typical Usage
-------------

::

    import mysql.connector

Documentation
-------------

The complete documentation for `mysql.connector` is available on `the official website <https://dev.mysql.com/doc/connector-python/en/>`_.

Errors with `mysql.connector.Error`
------------------------------------

The ``Error`` class is used to represent specific errors generated by the ``mysql.connector`` module.
