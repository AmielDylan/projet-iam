Function to retrieve the name of a substance
============================================

This function is used to retrieve the name of a substance from the database based on its ID.

Parameters:
-----------

- **idSubstance**: The ID of the substance.

Return Value:
-------------

- Returns the name of the substance.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
