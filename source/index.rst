Projet IAM Documentation
=========================

This document contains the documentation for the IAM project.

Introduction
------------

The IAM project is a platform for presenting drug interactions. It allows users to visualize interactions between two medications by entering their names.

Python Files
---------------

``app.py``
   This file contains the Flask routes for the web application's functionality.
   See detailed documentation: :doc:`app`.

``requetes.py``
   This file contains functions for database queries for autocompletion and medication verification.
   See detailed documentation: :doc:`requetes`.
