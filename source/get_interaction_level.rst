Function to retrieve the level of an interaction
===============================================

This function is used to retrieve the level of an interaction based on its ID.

Parameters:
-----------

- **idNiveau**: The ID of the interaction level.

Return Value:
-------------

- Returns the level of the interaction.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
