Documentation for index.html
=============================

This documentation provides an overview of the structure and elements defined in the 'index.html' file, which serves as the main HTML file for the project.

HTML Structure
--------------

- **DOCTYPE Declaration**:
    - Specifies the document type and version.

- **Language and Character Encoding Meta Tags**:
    - Define the language and character encoding for the document.

- **Title**:
    - Sets the title of the document.

External Dependencies
----------------------

- **Bootstrap CSS and JS**:
    - Links to the Bootstrap library for styling and functionality.

- **Google Fonts**:
    - Imports the Montserrat font family from Google Fonts.

- **Font Awesome Kit**:
    - Includes the Font Awesome icon library.

- **Custom CSS and JavaScript**:
    - Links to custom CSS and JavaScript files for additional styling and functionality.

Body Content
-------------

- **Header**:
    - Contains the project title, version information, and description.

- **Form Section**:
    - Allows users to input two medications and submit the form.

- **Autocomplete Input Fields**:
    - Provides autocomplete suggestions based on user input.

- **Class Selection (Dynamic)**:
    - Displays options for selecting medication classes if multiple classes are associated with the input medication.
    - The options are dynamically populated based on the selected medication.

- **Interaction Results (Dynamic)**:
    - Shows the details of medication interactions, including components, interaction level, class details, risks, and associated actions.
    - The results are dynamically generated and updated based on user input.

- **Footer**:
    - Displays copyright information.

JavaScript Dependencies
------------------------

- **Popper.js** and **Bootstrap JS**:
    - Required for dropdowns, tooltips, and other Bootstrap components.

- **jQuery**:
    - Used for DOM manipulation and event handling.

For more detailed information on each HTML element's purpose and usage, please refer to the code documentation and comments in 'index.html'.
