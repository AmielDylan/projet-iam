Function to retrieve the IDs of substances from a specialty
===========================================================

This function is used to retrieve the IDs of substances associated with a given specialty from the database.

Parameters:
-----------

- **specialite**: The name of the specialty.

Return Value:
-------------

- Returns a list of IDs of substances associated with the specialty.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
