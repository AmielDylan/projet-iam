Documentation for index.css
===========================

This documentation provides an overview of the CSS styles defined in the 'index.css' file, which are used to style the corresponding HTML elements in the project.

Body Styling
------------

- **html, body**: 
    - Sets the height of the HTML and body elements to 100%.
    - Specifies the font family and style for text content.
    - Defines a linear gradient background for the body element.

Container and Layout Styling
-----------------------------

- **.content**:
    - Specifies flex properties for content layout.

- **.footer**:
    - Ensures the footer element remains at the bottom of the page.

- **div.frame-1**:
    - Defines styling for a container with a flex layout.

- **div.container**:
    - Sets padding and margin properties for container elements.

- **div.third**:
    - Specifies border and border-radius properties for a specific container.

Typography and Text Styling
---------------------------

- **h1.title, h2.title-2, h3, p.version, p.warning, p.info, p.title-3, p.title-4**: 
    - Defines font properties for various headings and paragraphs.

- **p.end**:
    - Styles the end section with padding and color.

- **span.medName_yellow, span.medName_black**:
    - Styles specific spans with font weight and color.

Form and Input Styling
-----------------------

- **input, label.form-label, button**: 
    - Sets font properties and styles for input fields, labels, and buttons.

- **input.danger, input.warning, input.valide**:
    - Styles input fields with different border colors based on validation status.

Autocomplete Styling
---------------------

- **.autocomplete-item**:
    - Defines styling for autocomplete suggestions.

    - Specifies font properties, padding, and cursor style.

    - Adjusts font weight and color on hover for improved visibility.

Visibility and Media Queries
-----------------------------

- **.visible, .invisible**:
    - Controls the visibility of elements using the 'visibility' and 'display' properties.

- **@media (max-width: 767.98px)**:
    - Applies specific styles for responsive design on smaller devices.

For more detailed information on each style rule's implementation and usage, please refer to the code documentation and comments in 'index.css'.

For more details on each CSS style, please refer to :doc:`index.css`.
