Function to retrieve the ID of a medication class
=================================================

This function is used to retrieve the ID of a medication class from the database based on its name.

Parameters:
-----------

- **className**: The name of the medication class.

Return Value:
-------------

- Returns the ID of the medication class.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
