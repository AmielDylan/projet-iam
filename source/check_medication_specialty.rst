Function to check if a medication is a specialty
================================================

This function is used to determine whether a medication is a specialty by querying the database.

Parameters:
-----------

- **medicament**: The name of the medication to check.

Return Value:
-------------

- Returns a boolean value indicating whether the medication is a specialty.

Exceptions:
-----------

- Raises any error encountered during the database operation.

Note:
-----

Ensure that the database connection is properly established before calling this function.
